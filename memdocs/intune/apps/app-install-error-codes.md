# <a name="intune-app-installation-error-reference"></a>Intune 應用程式安裝錯誤參考

除了遵循提供的步驟以針對應用程式錯誤進行疑難排解以外，您也可以根據傳回的錯誤碼來了解特定應用程式錯誤。 當您找到相符的錯誤碼後，即可透過額外的描述與資訊來協助解決錯誤。

## <a name="android-app-installation-errors"></a>Android 應用程式安裝錯誤

本節會提及裝置系統管理員 (DA) 與 Samsung Knox Enrollment。 如需詳細資訊，請參閱[Android 裝置系統管理員註冊](../enrollment/android-enroll-device-administrator.md)與[使用 Samsung Knox Mobile Enrollment 自動註冊 Android 裝置](../enrollment/android-samsung-knox-mobile-enroll.md)。
