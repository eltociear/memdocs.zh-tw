# <a name="software-center-user-guide"></a>軟體中心使用者指南

適用於：  Configuration Manager (最新分支)

您組織的 IT 系統管理員會使用「軟體中心」來安裝應用程式、軟體更新，以及升級 Windows。 本使用者指南會為電腦使用者說明「軟體中心」的功能。

軟體中心功能的一般相關注意事項：

- 本文將說明「軟體中心」的最新功能。 如果您的組織使用舊版但仍受支援的「軟體中心」版本，則並非所有功能都可供使用。 如需詳細資訊，請連絡 IT 系統管理員。

- IT 系統管理員可能會停用「軟體中心」的某些功能。 您的特定體驗可能會有所不同。

- 如果有多個使用者同時使用一部裝置 (假設是透過多個遠端桌面工作階段)，則工作階段識別碼最低的使用者會是唯一能在軟體中心看到所有可用部署的使用者。 工作階段識別碼較高的使用者可能無法在軟體中心看到某些部署。 例如，工作階段識別碼較高的使用者可能會看到已部署的應用程式，但無法看到未部署的套件或工作順序。 同時，工作階段識別碼最低的使用者會看到所有已部署的應用程式、套件和工作順序。

## <a name="how-to-open-software-center"></a><a name="bkmk_open"></a> 如何開啟軟體中心

在 Windows 10 電腦上啟動「軟體中心」的最簡單方法就是按 [開始]  ，然後輸入 `Software Center`。 您不一定需要鍵入完整字串，Windows 就能找到最符合的結果。

如果您巡覽 [開始] 功能表，請在 [Microsoft 端點管理員]  群組底下尋找**軟體中心**圖示。

![Microsoft 端點管理員 [開始] 功能表圖示](media/microsoft-endpoint-manager-start-menu.png)

> [!NOTE]
> 1910 版的 [開始] 功能表路徑已變更。 在 1906 版和更早版本中，資料夾名稱為 **Microsoft System Center**。 在將 Configuration Manager 更新為 1910 版或更新版本時，請務必更新您維護的所有內部文件，以納入這些新的位置。

## <a name="applications"></a>應用程式

選取 [應用程式]  索引標籤，以尋找並安裝 IT 系統管理員部署給您或這部電腦的應用程式。

- **全部**：顯示您可以安裝的所有應用程式
- **必要**：IT 系統管理員會強制安裝這些應用程式。 如果您將這其中一個應用程式解除安裝，「軟體中心」將會重新安裝它。
- **篩選**：IT 系統管理員可能會建立應用程式類別。 如果可用，選取下拉式清單，即可將檢視篩選至僅限特定類別中的應用程式。 選取 [全部]  可顯示所有應用程式。
- **排序依據**：重新排列應用程式清單。 此清單預設會依 [最新]  排序。 最近提供的應用程式列出時會加上「新增」  標記，而且會顯示 7 天。
- **搜尋**：仍然找不到您要尋找的項目嗎？ 請在 [搜尋] 方塊中輸入關鍵字來尋找它！
- **切換檢視**：選取圖示，即可在清單檢視與並排顯示檢視之間進行切換。 應用程式清單預設會以圖形並排方式顯示。
    - 並排顯示檢視：IT 系統管理員可以自訂圖示。 每個圖格底下都會顯示應用程式名稱、發行者及版本。
    - 清單檢視：此檢視會顯示應用程式圖示、名稱、發行者、版本及狀態。

### <a name="install-multiple-applications"></a>安裝多個應用程式

<!-- 1357126 -->
您可以一次安裝多個應用程式，而不是等到安裝完一個，再開始安裝下一個。 並非所有應用程式都符合資格：

- 您可以看到該應用程式
- 該應用程式尚未下載或安裝
- IT 系統管理員不需要核准即可安裝該應用程式

一次安裝多個應用程式：

1. 若要在清單檢視中進入多重選取模式，請選取 [多重選取] 圖示 ![軟體中心多重選取圖示](media/software-center-multi-select-apps.png) 在右上角。
2. 選取清單中應用程式左邊的核取方塊，即可選取兩個或多個要安裝的應用程式。
3. 選取 [安裝所選項目]  按鈕。

應用程式正常安裝，只是現在會連續安裝。


## <a name="updates"></a>更新

選取 [更新]  索引標籤，以檢視並安裝 IT 系統管理員部署至這部電腦的軟體更新。  

- **全部**：顯示您可以安裝的所有更新
- **必要**：IT 系統管理員會強制安裝這些更新。
- **排序依據**：重新排列更新清單。 此清單預設會依 [應用程式名稱:  A 到 Z] 排序。

若要安裝更新，請選取 [安裝全部]  。

若只要安裝特定更新，選取進入多重選取模式的圖示。 核取要安裝的更新，然後選取 [安裝所選項目]  。


## <a name="operating-systems"></a>作業系統

選取 [作業系統]  索引標籤，以檢視並安裝 IT 系統管理員部署至這部電腦的 Windows 版本。  

- **全部**：顯示您可以安裝的所有 Windows 版本
- **必要**：IT 系統管理員會強制安裝這些升級。
- **排序依據**：重新排列更新清單。 此清單預設會依 [應用程式名稱:  A 到 Z] 排序。


## <a name="installation-status"></a>安裝狀態

選取 [安裝狀態]  索引標籤，以檢視應用程式的狀態。 您可能會看到下列狀態：

- **已安裝**：「軟體中心」已經在這部電腦上安裝此應用程式。
- **正在下載**：「軟體中心」正在下載要安裝在這部電腦上的軟體。
- **失敗**：「軟體中心」在嘗試安裝軟體時發生錯誤。
- **排程之後安裝時間**：顯示裝置下一個維護期間的日期和時間，用來安裝即將推出的軟體。 維護期間會由 IT 系統管理員定義。<!--1358131-->
    - 在 [全部]  和 [即將推出]  索引標籤中可以看到狀態。
    - 您可以在維護期間之前安裝，方法是選取 [立即安裝]  按鈕。


## <a name="device-compliance"></a>裝置相容性

選取 [裝置合規性]  索引標籤，以檢視這部電腦的合規性狀態。

選取 [檢查合規性]  ，以依據 IT 系統管理員所定義的安全性原則來評估此裝置的設定。


## <a name="options"></a>選項

選取 [選項]  索引標籤，以檢視這部電腦的額外設定。

### <a name="work-information"></a>工作資訊

指出您通常的工作時間。 IT 系統管理員可以排定在您的上班時間外進行軟體安裝。 請每天至少提供四小時來進行系統維護工作。 IT 系統管理員仍然可以在上班時間安裝重要的應用程式和軟體更新。

- 選取下拉式清單以選取您使用這部電腦的最早和最晚時間。 這些值預設會從 [上午 5 點]  到 [下午 10 點] 

- 選取您通常使用這部電腦之星期幾旁邊的核取方塊。 「軟體中心」預設只會選取非假日。  

指定您是否經常使用這部電腦工作。 您的系統管理員可能會自動安裝應用程式，或將其他應用程式提供給主要電腦使用。 <!--3485366-->

- 如果您使用的電腦為主要電腦，請選取 [我經常使用這部電腦工作]  。

### <a name="power-management"></a>電源管理

IT 系統管理員可以設定電源管理原則。 這些原則可協助您的組織在不使用這部電腦時節省電力。

若要讓這部電腦不受這些原則管理，請選取 [請勿在這部電腦套用我 IT 部門的電源設定]  核取方塊。 此設定預設為停用，電腦會套用電源設定。

### <a name="computer-maintenance"></a>電腦維護

指定「軟體中心」在期限之前套用軟體變更的方式

- **僅在指定的非工作時間自動安裝或解除安裝所需的軟體並重新啟動電腦**：預設會停用此設定。
- **當電腦在簡報模式時暫停軟體中心活動**：預設會啟用此設定。
- **同步處理原則**：當 IT 系統管理員指示您選取此按鈕時，請選取此按鈕。此電腦會向伺服器確認是否有任何新項目，例如應用程式、軟體更新或作業系統。

### <a name="remote-control"></a>遠端控制

指定您電腦的遠端存取和遠端控制設定

- **使用您 IT 部門的遠端存取設定**：預設會選取此核取方塊。
- **允許的遠端存取層級**：請從下列 3 個選項選取
    - **不允許遠端存取**
    - **僅限檢視**
    - **完整**：預設會啟用此層級。
- **允許系統管理員在我離開時對這部電腦進行遠端控制**。 根據預設，會將此設定設為 [是]  。
- **當系統管理員嘗試從遠端控制這部電腦時**：此設定有兩個選項
    - **每次都要求權限**：預設會選取這個選項。
    - **不要求權限**
- **遠端控制期間，顯示下列項目**：預設會同時選取這兩個選項
    - **通知區域中的狀態圖示**
    - **桌面上的工作階段連線列**
- **播放音效**：此設定有三個選項
    - **當工作階段開始及結束時**：預設會選取此設定。
    - **在工作階段期間重複提示**
    - **永不**

    如需詳細資訊，請參閱[遠端控制簡介](../clients/manage/remote-control/introduction-to-remote-control.md)
    

## <a name="custom-tab-in-software-center"></a>軟體中心內的自訂索引標籤

IT 系統管理員可能已在軟體中心內新增額外的索引標籤。 此索引標籤是由系統管理員命名，並指向其所指定的網站。 例如，您可能會有稱為「技術支援中心」的索引標籤，其指向您組織的技術支援中心網站。 <!--1358132-->
